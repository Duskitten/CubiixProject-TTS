### Scripts
- All variable names are written with no spaces, all lowercase
	- Y - "thisisavar"
	- N - "This_Is_A_Var"
- Function names are written with underscores, and all lowercase
	- Y - "this_is_a_function_name"
	- N - "thisIsAFunctionName"

### Folders
	- Folders will have "G_" as a prefix to keep them seperated
	- Capitalize First letter nouns in folder names
